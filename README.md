# 🎮 多人輪流專家猜測遊戲

## 🚀 快速開始

1. 安裝依賴：
   ```bash
   npm run setup
   ```

2. 啟動開發環境：
   ```bash
   npm run dev
   ```

3. 開啟瀏覽器訪問：
   - 前端：http://localhost:3000
   - 後端：http://localhost:5000

## 🎯 遊戲規則

- 最多8人同時遊玩
- 每人輪流當專家
- 專家根據其他人的提示猜測答案
- 猜對得分，提示者也獲得分數

## 🛠️ 技術棧

- 後端：Node.js + Express + Socket.io
- 前端：React + Socket.io-client + Styled-components
- 即時通訊：WebSocket
